Network diagrams

A network diagram is a pictorial representation of the connections between entities. One specific example is network diagram of daily commutes. Each city is represented as a node (circle) and an edge (line) between two nodes represents a connection. In some of the examples in this task, the nodes will be cities in the United Kingdom and edges will indicate that a large number of people commute between the cities by train.

In the example image, there are five nodes (A, B, C, D, and E). In this example, each node represents a city. The edges (lines) show popular commutes. By looking at the diagram, we can see that people commute between, for example, A & B, A & C, and A & E. No one, however, commutes between cities A & D (there is no edge).
